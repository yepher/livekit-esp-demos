# 05 – Walkie-Talkie (Push-to-Talk)

Two ESP32 devices in one LiveKit room with push-to-talk via the boot button. A complete, clonable app people can adapt for their own use cases.

## What You'll Learn

- Joining two ESP32 devices to the same LiveKit room
- Using the boot button as a push-to-talk trigger
- Managing audio publish/subscribe state at runtime
- Building a complete end-to-end app on the LiveKit ESP32 SDK
