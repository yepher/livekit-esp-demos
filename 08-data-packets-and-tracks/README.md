# 08 – Data Packets and Tracks

Sending structured data over LiveKit from an ESP32. Covers what is available today with data packets and the migration path to data tracks once they ship in the ESP SDK.

## What You'll Learn

- How to send and receive data packets from an ESP32 in a LiveKit room
- Structuring payloads for sensor readings, commands, and status updates
- What data tracks will offer once they land in the ESP SDK
- Planning a migration path from packets to tracks
